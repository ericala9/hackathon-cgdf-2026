# Créditos das Ilustrações

As ilustrações presentes na documentação deste projeto (pasta [`img`](img)) foram criadas com o auxílio de ferramentas de Inteligência Artificial Generativa.

- **Ferramenta:** Google Gemini 3
- **Modelo:** Nano Banana
- **Método:** Geração de imagens a partir de prompts descritivos originais elaborados pela autora da solução.

Estas imagens são utilizadas estritamente para fins ilustrativos e didáticos, visando facilitar a compreensão dos conceitos técnicos abordados no README.
